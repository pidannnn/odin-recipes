# Odin Recipes

Website resep sederhana yang dibuat sebagai bagian dari kursus Foundations di The Odin Project.

## Tentang Proyek

Proyek ini adalah proyek HTML pertama saya dari [The Odin Project](https://www.theodinproject.com/).  
Website ini memiliki halaman utama yang berisi link menuju beberapa halaman resep makanan.

Tujuan dari proyek ini adalah untuk melatih dasar-dasar HTML, seperti:

- Membuat struktur dasar HTML
- Menggunakan heading, paragraf, dan list
- Menambahkan gambar
- Membuat link antar halaman
- Mengatur struktur folder proyek dengan rapi

## Fitur

- Halaman utama dengan daftar resep
- Beberapa halaman resep
- Gambar pada setiap resep
- Daftar bahan dan langkah memasak

## Teknologi yang Digunakan

- HTML5

## Struktur Proyek

```bash
odin-recipes/
├── index.html
├── recipes/
    ├── pisang.html
    ├── roti.html
    └── terong.html
```
